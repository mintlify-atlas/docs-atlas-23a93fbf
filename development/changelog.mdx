## Release Process

Quick Select uses automated releases through GitHub Actions:

<Steps>
  <Step title="Version Bump">
    Run `npm run version` to update version numbers in `manifest.json` and `versions.json`
  </Step>

  <Step title="Tag Release">
    Create and push a version tag:
    ```bash
    git tag 1.5.1
    git push origin 1.5.1
    ```
  </Step>

  <Step title="Automated Build">
    GitHub Actions automatically:
    - Builds the plugin with Node.js 20
    - Runs TypeScript type checking
    - Creates a GitHub release
    - Uploads `main.js`, `manifest.json`, and `styles.css`
  </Step>
</Steps>
